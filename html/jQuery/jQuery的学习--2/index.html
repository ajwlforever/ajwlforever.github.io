三、jQuery内容选择器


1.：empty
var $div = $(“div:empty”);// 找既没有子元素又没有文本的指定元素
2.：parent
var $div = $(“div:parent”);// 找有子元素或有文本的指定元素
3.：contains(text)
var $div = $(“div:contains（’我是div’）”);// 找包含文本内容的指定元素 (是包含而不是等于)
4.：has
var $div = $(“div:has（’span’）”);// 找包含指定子元素的指定元素

四、属性和属性节点操作 attr prop

$(“span”).removeAttr(“class name”);
span的class和name属性全部删除。
